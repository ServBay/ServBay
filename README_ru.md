## Введение

Хватит тратить время на управление средой разработки!

ServBay — это мощный и комплексный инструмент для управления локальной средой веб-разработки, разработанный для профессиональных веб-разработчиков, теперь доступный как для **macOS, так и для Windows**. ServBay включает поддержку различных языков разработки, баз данных, веб-серверов, почтовых серверов, DNS-серверов и обратных прокси. Теперь он также обладает интегрированными возможностями разработки ИИ с помощью Ollama, объектным хранилищем с MinIO, мощным поиском с Typesense и Meilisearch, а также надежной системой резервного копирования. Все, что вам нужно для веб-разработки, здесь. Настройте профессиональную локальную среду разработки всего за 3 минуты.

![ServBay Dashboard](/images/dashboard.png)

## Возможности

### Управление несколькими сайтами

Управляете несколькими веб-сайтами одновременно? ServBay делает это легко. Без усилий настраивайте и запускайте множество веб-сайтов на вашем локальном компьютере, каждый со своими уникальными настройками, доменом и средой разработки. Интуитивно понятный интерфейс ServBay упрощает процесс, позволяя быстро переключаться между проектами и управлять их конфигурациями без сложной настройки или конфликтов.

![Multi-site Management](/images/website.png)

### Веб-серверы

ServBay включает в себя самые популярные веб-серверы — **Caddy, NGINX и Apache** — с поддержкой HTTP/3 и CORS. Он также поддерживает пользовательские домены и автоматическую настройку SSL, устраняя необходимость покупки доменов и SSL-сертификатов, что значительно экономит затраты во время разработки.

![Web Servers](/images/web-servers.png)

### Языки веб-разработки

ServBay поставляется с огромным набором языков разработки, включая **PHP** (5.6-8.5), **Node.js** (12-24), **Python** (2.7, 3.5-3.14), **Go** (1.11-1.24), **Java** (OpenJDK 7-24), **Ruby** (2.4-3.4), **Rust** и даже **.NET** (2.0-10.0) и **Mono**. Разработчики могут легко переключаться между различными версиями и использовать разные версии для разных проектов. Эта огромная гибкость дает разработчикам большую уверенность.

![Web Development Languages](/images/languages.png)

### Базы данных

Базы данных необходимы для веб-разработки. ServBay включает **MySQL**, **MariaDB** и все более популярный **PostgreSQL**. Также поддерживаются базы данных NoSQL: **Redis**, **Memcached** и **MongoDB** готовы к использованию «из коробки», не требуя настройки. ServBay также интегрирует **phpMyAdmin** и **Adminer** для удобного управления базами данных.

![Databases](/images/databases.png)

### Разработка ИИ с Ollama

Откройте для себя будущее разработки с интегрированными возможностями ИИ. ServBay включает Ollama, позволяя вам запускать мощные большие языковые модели (LLM), такие как Llama 3, Mistral и Gemma, локально. Создавайте и тестируйте приложения на основе ИИ с нулевой задержкой и полной конфиденциальностью данных, все это управляется через интуитивно понятный интерфейс ServBay.

![AI Development with Ollama](/images/ai.png)

### Мощный поиск с Typesense и Meilisearch

Обеспечьте свои приложения современными и быстрыми поисковыми системами. ServBay включает в себя как **Typesense**, так и **Meilisearch**, два ведущих решения для поиска с открытым исходным кодом, которые вы можете добавить в свои проекты одним щелчком мыши.

![Powerful Search with Typesense & Meilisearch](/images/search.png)

### Объектное хранилище с MinIO

Легко управляйте неструктурированными данными с помощью MinIO, высокопроизводительного, S3-совместимого сервиса объектного хранения, теперь включенного в ServBay. Он идеально подходит для хранения всего, от резервных копий и логов до медиафайлов и артефактов, предоставляя надежное решение для хранения для ваших локальных нужд разработки.

![Object Storage with MinIO](/images/minio.png)

### Доменные и DNS-сервисы

Домены — это расходуемый ресурс в жизненном цикле веб-разработки. ServBay включает DNS-сервер, позволяющий разработчикам использовать несуществующие домены и TLD без регистрации и даже выпускать для них SSL-сертификаты. Это не только значительно экономит затраты разработчиков, но и повышает безопасность — хакеры не могут получить доступ к несуществующему домену. Кроме того, ServBay предоставляет графический интерфейс для удобного управления вашим файлом hosts.

![Domain and DNS Services](/images/dns.png)

### Управление PKI и SSL-сертификатами

ServBay предоставляет систему PKI. Разработчики могут создавать свой собственный центр сертификации (CA) и выпускать SSL-сертификаты, что особенно полезно для небольших команд разработчиков. Используя частный CA, небольшие команды могут шифровать передачу данных и устанавливать внутреннее доверие в своей среде разработки. Это относится не только к веб-сервисам, но и к базам данных, SMTP и другим сервисам. ServBay даже поддерживает сертификаты S/MIME для шифрования электронной почты, сертификаты для подписи кода и сертификаты для подписи PDF-документов.

Конечно, ServBay также поддерживает получение и автоматическое продление SSL-сертификатов от Let's Encrypt, ZeroSSL и Google Trust Services через ACME.

![PKI and SSL Certificate Management](/images/ssl-pki.png)

### Почтовый сервер

Хотите использовать SMTP/POP3 в вашей локальной разработке? Нет проблем! ServBay включает встроенный почтовый сервер с **Mailpit** для локального тестирования электронной почты. Интегрированный с системой PKI, почтовый сервер поддерживает STARTTLS и SSL/TLS без настройки. Нужно пересылать почту на внешний SMTP-сервер? Почтовый сервер ServBay поддерживает пересылку. Мы также поддерживаем SpamAssassin, позволяя вам оценивать каждое исходящее письмо и снижать вероятность его пометки как спам.

![Mail Server](/images/email-server.png)

![Mail Server Webmail](/images/email-server-webmail.png)

### Обратный прокси

Нужно временно поделиться своим проектом с пользователями? Хотите разместить сервер дома, но у вас нет публичного IP-адреса? ServBay поможет вам. Он поддерживает **ngrok, frp, Cloudflared и Pinggy** для быстрого публичного доступа к вашему веб-сайту.

![Reverse Proxy](/images/tunnel.png)

### Среды выполнения на уровне проекта

ServBay предлагает настройку среды выполнения на уровне проекта. Вы можете настраивать и блокировать требуемые версии PHP и Node.js для разных проектов, делая разработку вашего проекта более гибкой и контролируемой.

![Project-Level Runtime Environments](/images/project-level-runtime.png)

### Мощное резервное копирование и восстановление

Безопасность данных и восстановление среды имеют решающее значение. ServBay имеет комплексную систему резервного копирования и восстановления, чтобы вы были спокойны.
*   **Резервное копирование в один клик и по расписанию**: Настройте автоматическое, запланированное резервное копирование или запускайте ручное резервное копирование ваших баз данных, файлов веб-сайтов, конфигураций служб и SSL-сертификатов.
*   **Простое восстановление и миграция**: Быстро восстановите вашу среду до предыдущего состояния в случае проблемы или используйте резервные копии для легкой миграции всей вашей установки ServBay на новый компьютер.

![Clean and Easy to Back Up](/images/easy-to-backup.png)

![Clean and Easy to Back Up](/images/easy-to-backup-2.png)

## Включенные пакеты

ServBay поставляется с различными необходимыми пакетами для оптимизации вашего процесса разработки:

-   **Языки программирования**: PHP (с 5.6 до 8.5-Alpha), Node.js (с 12 до 24), Python (2.7, 3.5-3.14), Golang (1.11-1.24), Java (OpenJDK 7-24), .NET (2.0-10.0), Mono (6.14), Ruby (2.4-3.4), Rust

-   **Другие инструменты**: phpMyAdmin, Adminer, Composer и другие

## Установка

ServBay доступен как для macOS, так и для Windows.

### macOS

1.  Загрузите последнюю версию ServBay для macOS с [официального сайта](https://www.servbay.com/download).
2.  Откройте установщик и следуйте инструкциям для завершения установки.
3.  Запустите ServBay и следуйте мастеру начальной настройки для его конфигурации.

### Windows

1.  Загрузите последнюю версию ServBay для Windows с [официальной страницы загрузки](https://www.servbay.com/download) или напрямую из [репозитория релизов для Windows](https://github.com/ServBay/ServBay-Windows-Release).
2.  Запустите установщик и следуйте инструкциям на экране.
3.  Запустите ServBay, чтобы начать настройку вашей локальной среды разработки.
